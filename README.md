# AI短剧/短视频生产效率工具

一个面向短剧、短视频生产流程的轻量化效率工具。

项目目标：把“主题 → 选题 → 脚本 → 分镜”的内容生产流程工具化，减少重复整理工作，提升团队内容策划与脚本生产效率。

## 功能列表

- 输入主题，生成 3 个短剧/短视频选题
- 选择选题后，生成三段式脚本
- 自动拆解为 5 个分镜镜头
- 支持 Markdown 方案导出
- 支持历史记录保存与恢复
- 当前版本使用 Mock 数据，后续可接入真实大模型 API

## 技术栈

- Python
- Streamlit
- JSON 本地存储
- Prompt 模板设计
- AI 辅助开发流程

## 运行方式

1. 安装依赖

```bash
pip install -r requirements.txt
```

2. 启动项目

```bash
streamlit run app.py
```

3. 浏览器打开 Streamlit 提供的本地地址。

## 项目亮点

- 贴近短剧/短视频业务场景
- 将内容生产流程拆解为可复用工具
- 支持结构化输出，便于后续导出、复用和团队协作
