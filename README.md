# AI短剧/短视频生产效率工具

这是一个面向短剧 / 短视频生产流程的 AI 辅助效率工具。

项目目标是把内容生产中的“主题输入 → 选题策划 → 脚本生成 → 分镜拆解 → 方案导出”流程工具化，减少重复整理工作，提升内容策划、脚本产出和团队协作效率。

当前版本为可演示 MVP，适合展示 AI 工具开发、业务需求拆解、Prompt 模板设计、轻量化工具搭建和快速迭代能力。

## 在线演示

项目已部署至 Streamlit Community Cloud。

可通过仓库右侧 Website 链接访问在线演示页面。

## 项目背景

短剧和短视频生产中，选题、脚本、分镜整理往往存在大量重复工作。

例如：

* 运营或编导需要反复整理选题方向
* 脚本结构需要统一格式
* 分镜信息需要重新拆分成表格
* 团队协作时需要可复制、可导出、可复用的内容方案

因此，本项目将上述流程拆解为一个轻量化工具，帮助使用者快速生成结构化内容方案。

## 核心功能

* 输入内容主题，生成 3 个短剧 / 短视频选题
* 每个选题包含标题、简介、爆点、适合平台
* 选择选题后，自动生成三段式脚本
* 将脚本拆解为 5 个分镜镜头
* 分镜包含画面描述、台词 / 旁白、时长、运镜建议
* 支持 Markdown 方案导出
* 支持历史记录保存与恢复
* 当前版本使用 Mock 数据，后续可替换为真实大模型 API

## 运行方式

安装依赖：

```bash
pip install -r requirements.txt
```

启动项目：

```bash
streamlit run app.py
```

启动后，在浏览器中打开 Streamlit 提供的本地地址即可使用。

## 项目亮点

### 1. 贴近真实业务场景

项目围绕短剧 / 短视频生产流程设计，不是单纯的技术 Demo，而是面向内容生产中的实际效率问题。

### 2. 流程完整

覆盖从主题输入、选题生成、脚本生成、分镜拆解到方案导出的完整链路，具备可演示性。

### 3. 输出结构化

选题、脚本、分镜均采用结构化数据组织，便于后续导出、复用、二次编辑和团队协作。

### 4. 便于接入真实大模型

当前版本使用 Mock 数据保证稳定演示，后续可将 services 层替换为 Claude、OpenAI、DeepSeek 等大模型 API 调用。

### 5. 具备迭代空间

后续可继续扩展 Prompt 模板管理、模型调用日志、Token 成本统计、Excel / Word 导出、多项目历史管理等功能。
